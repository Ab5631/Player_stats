VIRAT KOHLI CAREER STATISTICS ANALYSIS DASHBOARD

![Tumbnali](





This project showcases an interactive Power BI dashboard analyzing the cricketing career of Virat Kohli across ODI, T20, and Test formats. The dashboard provides a comprehensive view of his batting performance from 2008 to 2025, highlighting key career statistics, yearly trends, and milestone achievements.

1- Key Features
Career Overview: Total innings, runs, strike rate, and batting average.
Format Selection: Analyze performance across ODI, T20, and Test formats.
Runs by Year: Year-wise breakdown of total runs scored.
Boundaries Analysis: Distribution of fours and sixes across the career.
Milestone Tracking: Visualization of centuries (100s) and half-centuries (50s).
Interactive Filters: Dynamic year range and format selection for deeper insights.

2- Tools & Technologies
Power BI for data visualization and interactive reporting.
Data Transformation using Power Query for preprocessing.
DAX (Data Analysis Expressions) for calculated measures and KPIs.
